エンジン停止の理由

UFO や Bigfoot で車のエンジンが停止する理由

初出

【編】Steve Ward : Bigfoot や幽霊兵隊の出現で、車のエンジンが停止 (2021-12-03) UFO



前置き

- Steve Ward は経歴 50年（だったかの）ベテランの民間 UFO/怪奇現象 研究者で、下のインタビューでは Mothman 事件を中心に語っている。

- この動画は（私が視聴した時点では） live だったので、発言時刻が取れないし、音声書写（自動生成）も付与されていない。（後に live が取れ普通の動画になったが、音声書写（自動生成）はまだ付与されていない）



手抜き

- 車のエンジンが理由なしに停止し、始動できなくなった。その直後に Bigfoot が出現。Bigfoot が森の中へ去ると、エンジンが始動した。この現象は UFO でも頻繁に報告されている。

- 幽霊兵隊が出現、タクシーのエンジンが停止。


動画（）

MOTHMAN - CRYPTIDS - FLYING HUMANOIDS! ARE THEY REAL?



コメント1

- 民間の UFO 研究者も、 Bigfoot 研究者も以下の謎について長らく不思議がっている。

　　・なぜ、UFO/orb と Bigfoot でこのような奇妙なエンジン停止現象が共通するのか？

　　・なぜ、UFO/orb と Bigfoot が同時に目撃される事例がいくつも起きているのか？

　　・なぜ、Bigfoot がフラッシュ光とともに消え去る事例がいくつも起きているのか？

- Steve Ward も host の Dave Scott も、これらの謎があまりに奇妙で、全く理解不可能だ、お手上げだ、としている。


コメント2

- 彼らが揃って「これらの謎はあまりに奇妙で、全く理解不可能だ」とお手上げ状態になってしまう最大の原因は、ある共通した先入観にある。それは

　　(a) 「UFO/orb が」 エンジン停止を引き起こした。
　　(b) 「Bigfoot が」 エンジン停止を引き起こした。
　　(c) 「幽霊兵隊が」 エンジン停止を引き起こした。

　という先入観。

- 以下、私の推測だが、

　　・(1) UFO/Bigfoot 「が」エンジンを停止させたのではない。
　　・(2) 現場に異常な電磁場（*1）が発生し（仮説 1）、それがエンジンを停止させた。
　　・(3) その電磁場「によって」UFO/Bigfoot が出現した。
　　・(4) その電磁場「によって」missing-time が生じた。

　…のではないか。

- この (3), (4)  をより適切に表現すれば、

　　現場の電磁場が（エンジンのみならず）人間の脳神経系にも影響を与える（仮説 2）ことで

　　・(3)' 現場付近に UFO/Bigfoot が出現したという誤認識を生じさせた（仮説 3）。
　　・(4)' 記憶機能の中断によって、missing-time が起きたという誤解を生んだ（仮説 4）。

- 脳神経系に対する影響の度合いは個人差がある。ゆえに、同じ現場にいながら

　　・Bigfoot を見る人間と見ない人間がいる（事実）。
　　・見えた UFO の形状、色が人によって異なり、あるいは見ない人もいる（事実）。

- このような個人差もあるが、「見える」人々の間では共鳴作用や引き込み作用も生じるため（仮説 5）に、複数の人間が類似したもの（＝ UFO/Bigfoot）を体験する。その結果、それらが客観的な実在だと誤解する。

- さらに、影響を受けた脳神経系は一種の ESP 現象を引き起こす場合があり（仮説 6）、それが

　　・Bigfoot の足跡
　　・UFO の着陸痕
　　・UFO を撮影したつもりの写真に妙なボケが映り込む
　　・scoop mark
　　・Skinwalker ranch で多発した、自宅まで憑いてくる現象

　などなどを生じさせている。…のではないか。


コメント3

- 仮説 1-6 のうち、1 は多数の状況証拠がある。それ以外の 2-6 は実証的なデータに乏しいが、1-6 によって UFO/ET/abduction/幽霊/怪奇生物/超常現象/奇跡 といった謎の統一的な説明が可能となる。

- たとえば、以下のような一連の関連する謎について解明の見通しが見えてくる。

　　・なぜ、UFO/ET 遭遇や abduction 体験の後に、それらの体験者が様々な ESP 能力がついたと証言するのか。

　　・なぜ、UFO/ET 遭遇や abduction 体験の後に、それらの体験者の身の回りの電気機器が故障しがちになるのか。

　　・MIB や BEK の正体は何か。

　　・ET との hybrid とは何か。

　　・1800年代末の謎の飛行船の正体は何か。

　　・…





（*1）

- 実際のところ、この電磁場も＜未知の異常現象＞の副産物である可能性があるが、それを言い出すとキリがない。

(2021-12-03)
